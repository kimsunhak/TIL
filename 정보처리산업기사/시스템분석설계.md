## 과목 : 시스템 분석 설계

> #### 평균수리시간(MTTR)
>
> 계산법 : (고장1 + 고장2 + 고장3) / 고장갯수
>
> 
>
> #### 평균고장간격(MTBF)
>
> 계산법 : (가동1 + 가동2 + 가동3) / 가동갯수
>
> 
>
> #### 신뢰도
>
> 계산법 : MTBF / (MTTR + MTBF)



> #### **색인 순차 파일(Indexed Sequential File)에서 색인 영역(index area)의 종류**
>
> Master Index area, Cylinder Index area, Track Index area
